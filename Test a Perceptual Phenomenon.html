We will set significance level "alpha" as 5% (0.05). We will perform statistical test (one-tailed test) by using python scipy.stats library.

Since we will try to reject the null hypothesis in favor of the alternative hypothesis we must set the ttest_ind function first argument as TC (congruent data from sample) and second argument as TI (incongruent data from sample) so that Tc > Ti or Tc - Ti > 0 can be proved.

In [24]:
sta, pval = sp.ttest_ind(TC, TI)
In [25]:
print(sta)

In [26]:
print(pval)

4.5949489622951854e-08

The test above shows us that statistic is -6.53 and p-value is 4.60e-08.

We must divide pval between 2 and significance level "alpha" between 2 to get one-tailed test because ttest_ind function results are associated to two-tailed test. If (pval/2 < 0.025) then we've failed to reject the null hypothesis in favor of alternative hypothesis.

I think delayed time reaction to incongruent words is due to human habits of associating objects or pictures to words; so in this case the picture (word color) tells our brains something different from what we are reading leading to confusion. A similar effect would be produced by numerical Stroop effect in which a digit is presented as big or small irrespective of its numerical value, then comparison of digits is carried out (incongruent trials leads to slower responses "let's say 7 is written in a smaller font size than 2" than congruent trials "let's say 2 is written in a smaller font size than 7).
